# 🌱 KiwiSense

An open-source ESP32-based IoT monitoring system designed to monitor young kiwi plants and support data-driven irrigation decisions.

---

## Project Overview

KiwiSense is a modular smart agriculture project built to monitor environmental conditions around kiwi plants.

The objective is to create a reliable, low-cost, and expandable monitoring platform that can later automate irrigation and run entirely on solar power.

---

## Version 1 Goals

- Soil Moisture Monitoring
- Soil Temperature Monitoring
- Air Temperature
- Air Humidity
- OLED Display
- ESP32 Controller

---

## Hardware

- ESP32 NodeMCU (ESP-WROOM-32)
- Capacitive Soil Moisture Sensor
- DS18B20 Waterproof Temperature Sensor
- DHT22 Temperature & Humidity Sensor
- 0.96" OLED Display (SSD1306)
